# Data

This project uses the **KG20C** dataset from [Hung-Nghiep Tran's repository](https://github.com/tranhungnghiep/KG20C), which is derived from the **Microsoft Academic Graph**.

## Required Files

To run the pipeline, download the following files from the KG20C repository and place them in `data/kg20c/`:

- `all_entity_info.txt`
- `all_relation_info.txt`
- `train.txt`
- `valid.txt`
- `test.txt`

## Directory Structure

```
data/
├── kg20c/          # Raw KG20C data (place downloaded files here)
├── csv/            # Generated by step 1: TSV → CSV
├── processed/      # Generated by step 2: Embeddings
└── sqlite/         # Generated by step 3: Database
```
